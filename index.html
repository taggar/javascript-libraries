Javascript Libraries

Showcasing a sampling of javascript libraries.

  • Lightbox: a library to show images (or possibly other content as well) in an overlay on the page, including navigation when images are declared to be part of a set. Fancybox is a jQuery alternative.
  • Particles: a library to show an animated set of particles (or other shapes) that react to mouse position and movement. The parameters (numeber of particles, shapes, behaviour, ...) is determined by a configuration file expressed in JSON format.
  • Wow: a library to reveal animations when you scroll. Easily customize animation settings: style, delay, length, offset, iterations...

As a bonus, we know that we might need to write about code, in which case a library to highlight bits of code comes in handy, so we also look into a few highlighting libraries.
